## Specifiche tecniche:
- Formato: JPG o PNG
- Qualità: Alta risoluzione (almeno 1920px sul lato maggiore)
- Ottimizzazione: Compressa per il web ma mantenendo qualità
- Stile: Luminose, naturali, colori caldi
- Mood: Professionale ma accogliente, focus su natura e benessere

## Come aggiungere le immagini:
1. Salvare le immagini in questa cartella (/public/images/)
2. Aggiornare i percorsi nel codice sostituendo "/api/placeholder/" con "/images/"
3. Assicurarsi che i nomi dei file corrispondano a quelli specificati nel codice
